# TPVFácil

Software TPV para hostelería y comercio. Cumple Verifactu (AEAT). Pago único, sin suscripciones.

## Estructura

```
/app   → Aplicación de escritorio (Java 21 + JavaFX)
/web   → Sitio web corporativo (HTML/CSS/JS)
```

**Empieza siempre por `CLAUDE.md`.**

## Requisitos mínimos del usuario final

- Windows 10 / 11 (64 bits)
- 4 GB RAM
- 500 MB espacio en disco
- Conexión a internet (para Verifactu)
